Task Scenario: Add Classes to Profile

Mike is in his junior year of college and wants to switch his major after deciding he wants to pursue a career in HCI. Only problem, however, is that the courses are much more intense! So, he decides he needs to start working with others more often. Mike hears about an app that can help him find others to study with at anytime, decides to download it, and input his classes so he can get started.


ESSENTIAL USE CASE

User's Purpose System Response
Enter search for current courses Show search results
Select proper course Add course to user profile

CONCRETE USE CASE

User's Purpose System Response
The student enters search parameters for the course; class name, course number. Show search results
The student selects registered course from search results. Display feedback of verification that course was added to profile.

Task Scenario: Create Study Session

It's now midterm time on campus and Mike does not feel as comfortable with his class material as he would like. Unfortunately, he doesn't know anyone in his classes but still needs to find a way to get a study session going. Since his classes are already added to the study session app, he decides to use it to get a study session started and attract others looking to study as well.


ESSENTIAL USE CASE

User's Purpose System Response
Select create group button Display current courses
Select course for study group Display potential study locations
Select study location Display “complete session creation” button
Select “Complete Session Creation” button Display Study Session creation confirmation

CONCRETE USE CASE

User's Purpose System Response
The student selects create group button at main screen The system displays all of the student’s courses in their profile
The student selects the course they want to start a study session for The system displays all possible locations where the study session can take place
The student selects the desired location for their study session The system displays a “complete study session creation” button
The student selects the “complete study session creation” button The system displays confirmation that the study session was created
